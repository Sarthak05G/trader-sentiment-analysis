# Trader Performance vs Market Sentiment Analysis

## Objective
Analyze how Bitcoin Fear/Greed sentiment impacts trader behavior and performance on Hyperliquid.

---

## Methodology
- Cleaned and aligned datasets at daily level
- Engineered metrics such as daily PnL, win rate, trade frequency, leverage usage
- Compared performance across Fear vs Greed regimes
- Segmented traders by leverage, activity, and consistency
